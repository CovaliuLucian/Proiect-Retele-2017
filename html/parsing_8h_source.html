98  // read a token.
99  // if the token is a number, then push it to the output queue.
100  // if the token is an operator, then:
101  // while (there is an operator at the top of the operator stack with
102  // greater precedence) or (the operator at the top of the operator stack has
103  // equal precedence and
104  // the operator is left associative) and
105  // (the operator at the top of the stack is not a left bracket):
106  // pop operators from the operator stack, onto the output queue.
107  // push the read operator onto the operator stack.
108  // if the token is a left bracket (i.e. "("), then:
109  // push it onto the operator stack.
110  // if the token is a right bracket (i.e. ")"), then:
111  // while the operator at the top of the operator stack is not a left bracket:
112  // pop operators from the operator stack onto the output queue.
113  // pop the left bracket from the stack.
114  // /* if the stack runs out without finding a left bracket, then there are
115  // mismatched parentheses. */
116  // if there are no more tokens to read:
